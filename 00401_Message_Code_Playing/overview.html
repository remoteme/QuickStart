Send Message device to device

Sometimes You want to send some data into your ESP or WebPage, With RemoteMe You can do it simple using userMessages. In this example WebPage is sending string and double value in one user message. And ESP receive message decode it and print using serial. Then esp modifies recevied string and double and send it back. At the webPage You can receive this message and see total time of communication.
By default all User messages goes through RemoteMe but by uncommenting some marked code in ESP You can turn on direct connection. With direct connection if webPage and ESP is in the same local network all messages will be send directly to you ESP.
For code documentation please visit Source code documantation in RemoteMe
It's for both ESP and RasbperryPi but both are not required You can install program only at one of the device
